## 求叶子节点个数
```c++
    //计算叶子个数
    int calculeaf(t *root) {
        if (root == NULL) {
            return 0;
        }
        if (root->left == NULL && root->right == NULL) {
            return 1;
        }
        else {
            return calculeaf(root->left) + calculeaf(root->right);
        }
    }
```
