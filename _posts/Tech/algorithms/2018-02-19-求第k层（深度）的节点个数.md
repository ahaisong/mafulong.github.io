## 求第k层（深度）的节点个数
```c++

    //求第k层（深度）的节点个数
    //求k层个数即求k-1层孩子个数，第一层就一个节点，就是1
    int levelcount(t* root, int k){
        if (root == NULL || k < 0) {
            return 0;
        }
        if (k == 1) {
            return 1;
        }
        int l = levelcount(root->left, k - 1);
        int r = levelcount(root->right, k - 1);
        return l + r;
    }
```
