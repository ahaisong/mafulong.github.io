## 计算深度
```c++
    // 计算深度
    int calcudepth(t *root) {
        if (root == NULL) {
            return 0;
        }
        int ldepth = calcudepth(root->left);
        int rdepth = calcudepth(root->right);
        return ldepth > rdepth ? ldepth+1 : rdepth+1;
    }
```
